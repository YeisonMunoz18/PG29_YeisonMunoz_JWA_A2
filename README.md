----------
Game + Level Editor Implementation. Users can add, move, and delete level elements (blocks, catapults, enemies, birds, and goals) on a canvas, then save, load, and delete levels.
Then, you can play your created levels in the game. This game called Super Mad Flying Creatures consists in hitting the enemys with a bird, knock them all down and you win the level.

**Download/Install**
---------
 - Clone Repository and run with cmd (npm run dev) in the local host. Then, open in your browser.

**How to use**
--------

*LEVEL EDITOR*

The Level Editor will allow you to create a simple game with basic shapes that represents differentes elements in the level. Elements such as:
- Blocks
- Catapults
- Enemies
- Birds
- Goal

You can create your game just using the selector and moving your elements wherever you want. You can save it and load it with a certain ID you want to put.
Also, if you want to delete an element, right click and delete the single element, but if you want to delete the entire level, use the delete button with the id of the level you want to delete.

*GAME*

The game section will let you play all the levels you created in the previous editor. After the level creation, open the Game file (html) and it will run automatically your levels!
